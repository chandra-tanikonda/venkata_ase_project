Project Description

This project is designed to take down employees’ attendance using a QR code. Every employee will have a card that contains a unique QR code representing their ID. Employees scan their card in front of a webcam upon arrival, and the system notes down their attendance, including the date and time. The system stores all the employees' attendance records and generates overall attendance reports in an Excel sheet for the admin. It also allows the admin to search for specific employees’ attendance.
