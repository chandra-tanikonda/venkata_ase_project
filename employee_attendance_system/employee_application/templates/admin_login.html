Project Description

This project is designed to take down employees’ attendance using a QR code. Every employee will be given a card containing a unique QR code representing their unique ID. They can scan their cards in front of a webcam upon arrival, and the system will record their attendance with date and time. The system stores all employee attendance records and generates an overall report in an Excel sheet for the admin. It also allows the admin to search for specific employees' attendance.
